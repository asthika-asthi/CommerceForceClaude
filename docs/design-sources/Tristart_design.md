## Design Patterns

### Color Scheme
- **Overall Scheme:** Light, Modern, Professional
- **Background:** White (#FFFFFF)
- **Primary Text:** Black (#000000)
- **Primary Accent:** Deep Navy Blue (#293897)
- **Secondary Accent:** Bright Blue (#007BFF)
- **Secondary Text:** Gray (#6C757D)

### Visual Hierarchy
1. **Primary:** Deep Navy Blue (#293897) - Main CTAs, important elements
2. **Secondary:** Bright Blue (#007BFF) - Highlights, secondary actions
3. **Tertiary:** Gray (#6C757D) - Supporting information
4. **Text:** Black on white for maximum readability

### Consistency Guidelines
- Use Oswald for all headings and large display text
- Use Roboto for body text and descriptions
- Maintain 0px border radius for sharp, modern edges
- Deep Navy (#293897) as primary action color
- Bright Blue (#007BFF) as secondary accent
- No drop shadows on buttons (flat design)
- Minimum padding: 12px on all interactive elements
- Consistent spacing using 4px base unit

---

## Accessibility

### Color Contrast
- Deep Navy (#293897) on White: 9.2:1 ratio ✓ (WCAG AAA)
- Bright Blue (#007BFF) on White: 5.3:1 ratio ✓ (WCAG AA)
- Black (#000000) on White: 21:1 ratio ✓ (WCAG AAA)
- Gray (#6C757D) on White: 6.3:1 ratio ✓ (WCAG AA)

### Typography
- Oswald and Roboto both chosen for excellent readability
- Minimum font size: 12px for captions, 16px for body
- Line height: 1.2-1.6 for readability
- Sufficient letter spacing for clarity

### Interactive Elements
- Minimum touch target size: 44x44px
- Focus states clearly visible (outline or background change)
- Keyboard navigation fully supported
- Alt text required for all images

## Responsive Design

### Mobile First Approach
- Base styles designed for mobile (< 768px)
- Progressive enhancement for larger screens
- Flexible grid system with proper breakpoints

### Breakpoints
```
Mobile: 0px - 767px
  - Single column layout
  - Full-width buttons
  - Simplified navigation
  
Tablet: 768px - 1023px
  - 2-3 column layout
  - Half-width elements
  - Hybrid navigation
  
Desktop: 1024px and above
  - Full multi-column layout
  - Optimized for large screens
  - Full navigation menu
```

### Responsive Typography
```
H1:
  Mobile: 36px
  Tablet: 48px
  Desktop: 60px

H2:
  Mobile: 24px
  Tablet: 28px
  Desktop: 33px

Body:
  Mobile: 14px
  Tablet: 15px
  Desktop: 16px
```
